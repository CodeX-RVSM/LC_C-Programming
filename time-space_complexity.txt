Time Complexity -
Time complexity measures how fast an algorithm runs, based on the size of the input.
Notation Name Example
O(1)         Constant Accessing an array element: arr[5]
O(log n) Logarithmic Binary search
O(n)         Linear         Loop through array(linear Search)
O(n log n) Linearithmic Merge sort, Quick sort (avg)
O(n²)         Quadratic Nested loops (e.g., bubble sort)
O(2ⁿ)         Exponential Recursive Fibonacci (naive)
O(n!)         Factorial Generating all permutations

Notation Example
